# Data Analyst Handbook

> Справочник по профессии Data Analyst — от установки инструментов до A/B-экспериментов. Это **справочник, а не курс**: открывай нужную тему и бери, что надо, — страницы самодостаточны.

🔗 **Сайт:** https://da-handbook.trosman1999.workers.dev

## Что внутри

Справочные страницы с примерами кода, разбором типичных ошибок и разобранными задачами:

- **SQL** — от `SELECT` до оконных функций, CTE и оптимизации
- **Excel / Google Sheets**, **Python** (pandas, визуализация)
- **Статистика** и **A/B-тестирование**
- **Визуализация данных** и **BI** (Tableau, Power BI, Looker)
- **Продуктовая аналитика**, **ML-база**, **современный data-стек**
- **Карьера**: резюме, портфолио, собеседования

Язык контента — русский, термины и код — английские.

## Технологии

Статический сайт на [Astro](https://astro.build/) + [Starlight](https://starlight.astro.build/),
поиск через Pagefind, хостинг на Cloudflare. Тёмная тема, диаграммы и формулы.
