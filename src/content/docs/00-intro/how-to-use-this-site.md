## Структура

Каждый раздел в sidebar слева — это блок знаний. Внутри раздела страницы пронумерованы и идут от простого к сложному. Можно читать подряд или прыгать по темам.

## Поиск

`Cmd/Ctrl + K` — открывает поиск по всему сайту. Ищет по заголовкам и содержимому страниц.

## Обозначения в тексте

:::tip[Совет]
Совет или хорошая практика.
:::

:::caution[Осторожно]
Типичная ошибка или подводный камень.
:::

:::note[Заметка]
Дополнительная информация, не критичная для понимания.
:::

:::danger[Внимание]
Что-то, что точно ломает данные или анализ.
:::

## Задачи для практики

В конце многих страниц есть раскрывающиеся блоки с разобранными задачами — сначала попробуй ответить сам, потом раскрой решение.
