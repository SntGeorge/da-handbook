:::tip[Коротко]
**VS Code** — бесплатный редактор, в котором аналитик пишет и Python-скрипты, и Jupyter-ноутбуки в одном окне. Поставь сам редактор, расширения **Python** и **Jupyter** — и можно открывать `.ipynb` прямо в нём.
:::

## Установка VS Code

Скачай с [code.visualstudio.com](https://code.visualstudio.com/) и установи как обычное приложение. Бесплатно, есть под все ОС.

## Обязательные расширения

Открой панель Extensions (`Cmd/Ctrl + Shift + X`) и поставь:

| Расширение | Зачем |
|------------|-------|
| **Python** (Microsoft) | Запуск, отладка, выбор интерпретатора |
| **Jupyter** (Microsoft) | Ноутбуки `.ipynb` прямо в редакторе |
| **Pylance** | Автодополнение и проверка типов (ставится с Python) |

## Jupyter в VS Code

Отдельно ставить ничего не нужно — расширение Jupyter подтянет ядро. Если запускаешь ноутбук впервые, VS Code сам предложит установить пакет:

```bash
pip install jupyter
```

Создай файл с расширением `.ipynb`, открой его — справа вверху выбери интерпретатор Python, и можно запускать ячейки (`Shift + Enter`).

:::note[Выбор интерпретатора]
Если используешь [виртуальное окружение](/01-setup/virtual-environments/), не забудь выбрать его интерпретатор: `Cmd/Ctrl + Shift + P` → «Python: Select Interpreter». Иначе библиотеки из venv не подхватятся.
:::

## Альтернатива: JupyterLab

Если хочется чистый ноутбучный интерфейс без редактора:

```bash
pip install jupyterlab
jupyter lab
```

Откроется в браузере. Для большинства задач VS Code удобнее (код + ноутбуки + терминал в одном месте), но это дело вкуса.
