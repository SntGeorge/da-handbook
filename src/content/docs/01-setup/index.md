## Что внутри

- **Python** — установка, версии, виртуальные окружения.
- **VS Code и Jupyter** — редактор, расширения, ноутбуки.
- **PostgreSQL и DBeaver** — локальная БД для практики SQL и GUI-клиент.
- **Git и GitHub** — версионирование с нуля.
- **Командная строка** — необходимый минимум для аналитика.
