## План страницы

- BETWEEN ... AND
- IN (...) и NOT IN — особенно с NULL
- LIKE и подстановки % и _
- ILIKE (Postgres) для регистронезависимого поиска
- IS NULL / IS NOT NULL — почему = NULL не работает
- Регулярные выражения (~)
- Задачи
