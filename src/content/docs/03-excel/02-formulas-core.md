:::tip[Коротко]
Рабочая лошадка аналитика в Excel — **`SUMIFS` / `COUNTIFS` / `AVERAGEIFS`**: посчитать сумму/количество/среднее по условиям (аналог `GROUP BY ... WHERE` из SQL). Плюс `IF` для логики и `ROUND` для аккуратных чисел. С этим набором закрывается большинство задач.
:::

## Зачем это нужно

«Сколько продали в марте по категории "Электроника"?» — это `SUMIFS`. «Сколько заказов из России?» — `COUNTIFS`. Эти функции заменяют ручную фильтрацию и сводят к одной формуле то, что иначе делалось бы мышкой.

```text title="Демо-данные (A1:D6)"
country  category      status   amount
RU       Электроника   paid     2500
RU       Книги         paid     1800
KZ       Электроника   paid     4200
KZ       Книги         cancel   700
DE       Электроника   paid     3000
```

## IF и логика

`IF(условие; если_да; если_нет)` — ветвление:

```text
=IF(D2>2000; "крупный"; "мелкий")     → "крупный"
```

Несколько условий — через `IFS` (без вложенности) или `AND`/`OR`:

```text
=IFS(D2>=3000;"A"; D2>=2000;"B"; TRUE;"C")
=IF(AND(C2="paid"; D2>2000); "ок"; "нет")
```

| amount | status | IF(AND...) |
|--------|--------|------------|
| 2500   | paid   | ок         |
| 1800   | paid   | нет        |
| 700    | cancel | нет        |

## SUMIFS / COUNTIFS / AVERAGEIFS

Главные функции. Синтаксис: сначала диапазон для подсчёта, потом пары «диапазон условия; условие».

```text
=SUMIFS(D:D; A:A;"RU")                    сумма по России       → 4300
=SUMIFS(D:D; A:A;"KZ"; C:C;"paid")        Казахстан и paid      → 4200
=COUNTIFS(A:A;"RU")                        число заказов RU      → 2
=AVERAGEIFS(D:D; B:B;"Электроника")        средний чек категории → 3233
```

:::note[SUMIF vs SUMIFS]
Старый `SUMIF` берёт одно условие и порядок аргументов другой (`SUMIF(диапазон_условия; условие; диапазон_суммы)`). `SUMIFS` поддерживает много условий и порядок удобнее. **Используй всегда `SUMIFS`** — даже для одного условия, чтобы не путаться.
:::

## Округление и числа

```text
=ROUND(D2/3; 2)      округлить до 2 знаков     → 833,33
=ROUNDUP(...);  =ROUNDDOWN(...)                вверх / вниз
=INT(7.9)            отбросить дробь           → 7
=MOD(7; 3)           остаток от деления        → 1
```

:::caution[Условия с подстановкой и сравнением]
В условиях `SUMIFS`/`COUNTIFS` можно писать `">2000"`, `"<>paid"`, `"Эл*"` (звёздочка — любой текст). Сравнение и шаблон оборачивай в кавычки: `COUNTIFS(D:D;">2000")`, а не `>2000`.
:::

## Когда что применять

| Задача | Функция |
|--------|---------|
| Сумма по условиям | `SUMIFS` |
| Количество по условиям | `COUNTIFS` |
| Среднее по условиям | `AVERAGEIFS` |
| Ветвление логики | `IF` / `IFS` |
| Несколько условий в одном `IF` | `AND` / `OR` |

## Задачи для самопроверки

<details>
<summary>1. Как посчитать выручку только по оплаченным заказам из Казахстана?</summary>

`=SUMIFS(D:D; A:A;"KZ"; C:C;"paid")`. Первый аргумент — что суммируем (amount), дальше пары «столбец условия; значение». Ответ на демо-данных: 4200.

</details>

<details>
<summary>2. Чем `SUMIFS` удобнее `SUMIF`?</summary>

`SUMIFS` поддерживает несколько условий и имеет логичный порядок аргументов (сначала диапазон суммы). `SUMIF` — только одно условие и обратный порядок. Чтобы не держать в голове два синтаксиса, используй везде `SUMIFS`.

</details>
