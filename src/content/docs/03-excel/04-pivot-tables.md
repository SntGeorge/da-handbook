:::tip[Коротко]
Сводная таблица (Pivot Table) — это `GROUP BY` мышкой: перетаскиваешь поля в области, и Excel сам считает суммы, количества и средние по группам. Главный инструмент быстрого анализа без единой формулы.
:::

:::note[Поток данных]
Вход: плоская таблица строк
→ Обработка: перетаскиваешь поля в области (строки/столбцы/значения) — Excel группирует и агрегирует
→ Выход: сводная таблица «метрика по разрезам».
Зачем: получить сводку мышкой, без формул — это `GROUP BY` для Excel.
:::

## Зачем это нужно

Когда нужно за минуту получить «выручку по странам и месяцам», писать `SUMIFS` для каждой ячейки долго. Сводная строит такую таблицу перетаскиванием полей и пересобирается на лету.

```text title="Исходные данные"
date         country   category      amount
2026-01-05   RU        Электроника   2500
2026-01-12   RU        Книги         1800
2026-02-03   KZ        Электроника   4200
2026-02-20   KZ        Книги         700
2026-02-25   DE        Электроника   3000
```

## Создание

1. Выдели данные (с заголовками) → вкладка **Вставка → Сводная таблица**.
2. Excel предложит новый лист — соглашайся.
3. Справа появится панель с четырьмя областями.

## Четыре области

| Область | Что кладём | Роль |
|---------|-----------|------|
| **Строки** (Rows) | `country` | группы по вертикали |
| **Столбцы** (Columns) | `category` | группы по горизонтали |
| **Значения** (Values) | `amount` | что считаем (сумма/кол-во/среднее) |
| **Фильтры** (Filters) | `date` | отбор всей таблицы |

Перетащив `country` в строки и `amount` в значения, получишь:

| country | Сумма amount |
|---------|--------------|
| RU      | 4300         |
| KZ      | 4900         |
| DE      | 3000         |

:::caution[Сумма или количество?]
Кинул в «Значения» текстовое поле — Excel посчитает «Количество». Кинул число — «Сумму». Если видишь не ту операцию, кликни по полю в области значений → **Параметры поля значений** и выбери нужную (Сумма, Количество, Среднее, Максимум…).
:::

## Группировка по датам

Брось `date` в строки, кликни правой → **Группировать** → выбери «Месяцы»/«Кварталы»/«Годы». Excel свернёт ежедневные даты в нужный период — основа отчётов «по месяцам».

## Вычисляемые поля

Если нужна метрика, которой нет в данных (например, средний чек = сумма / количество), не обязательно править исходник: **Анализ сводной → Поля, элементы и наборы → Вычисляемое поле** и задай формулу.

## Срезы (Slicers)

Срез — это кнопочный фильтр поверх сводной: **Анализ → Вставить срез**, выбираешь поле (например, `country`), и кликами фильтруешь таблицу. Удобно для мини-дашбордов; один срез можно подключить к нескольким сводным.

<details>
<summary>1. Хочу выручку по странам в строках и категориям в столбцах. Что куда?</summary>

`country` → область «Строки», `category` → «Столбцы», `amount` → «Значения» (операция «Сумма»). Получишь матрицу страна × категория с суммами на пересечениях.

</details>

<details>
<summary>2. В сводной вместо суммы показывается количество. Почему?</summary>

Скорее всего, поле попало в «Значения» как текст, либо операция выставлена в «Количество». Открой параметры поля значений и переключи на «Сумма». Также проверь, что в столбце нет текста среди чисел.

</details>
