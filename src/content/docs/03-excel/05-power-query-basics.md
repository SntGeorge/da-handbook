:::tip[Коротко]
Power Query — встроенный в Excel ETL-инструмент: загружает данные из файлов/баз/веба, чистит и преобразует их **по записанным шагам**, а потом обновляет результат одной кнопкой. В отличие от формул, шаги повторяются автоматически при новой выгрузке.
:::

## Зачем это нужно

Если каждый месяц приходит новый CSV, который надо одинаково почистить, — формулами это ад. Power Query запоминает цепочку преобразований один раз; в следующий раз просто кладёшь новый файл и жмёшь **Обновить**. Это разница между «делать руками каждый раз» и «настроить однажды».

## Где находится

Вкладка **Данные → Получить данные** (Get Data). Откроется редактор Power Query — отдельное окно, где слева список запросов, справа — записанные шаги (Applied Steps).

## Импорт

Power Query тянет данные откуда угодно:

- **CSV / текст** — Данные → Из текста/CSV.
- **Excel-файл или папка с файлами** — можно собрать все файлы из папки разом.
- **База данных** — PostgreSQL, SQL Server и др.
- **Веб** — таблицы прямо со страницы по URL.

## Базовые трансформации

Всё делается кликами, а Power Query записывает шаги:

- удалить/переименовать столбцы;
- изменить тип данных (текст → число → дата);
- фильтровать строки, убрать дубликаты;
- разделить столбец по разделителю, заменить значения;
- развернуть/свернуть (pivot/unpivot).

:::note[Каждый шаг записывается]
Список «Применённые шаги» справа — это история твоих действий. Любой шаг можно отредактировать или удалить, и Power Query пересчитает результат. Это и делает обработку воспроизводимой.
:::

## Merge и Append

Два способа объединить запросы — их часто путают:

| Операция | Что делает | Аналог в SQL |
|----------|-----------|--------------|
| **Merge** | соединяет таблицы по ключу (добавляет столбцы) | `JOIN` |
| **Append** | подклеивает строки одной таблицы под другую | `UNION ALL` |

Merge — когда надо подтянуть данные из справочника. Append — когда есть выгрузки за разные месяцы с одинаковыми столбцами и нужно сложить их в одну.

## M-язык

Под капотом каждый шаг — это код на языке **M**. Обычно его не пишут руками: кликов хватает. Но строку формул видно, и иногда удобно поправить шаг прямо в коде. Глубоко учить M на старте не нужно.

## Обновление данных

Кнопка **Обновить всё** (Данные → Обновить всё) прогоняет все шаги заново на свежих данных. Положил новый файл с той же структурой → обновил → готово.

:::caution[Структура должна совпадать]
Power Query привязан к именам и типам столбцов источника. Если в новом файле переименовали столбец или сменили формат даты — шаги «поедут». Держи структуру выгрузок стабильной или закладывай это в шаги.
:::
