:::tip[Коротко]
Google Sheets умеет почти всё то же, что Excel, но имеет несколько уникальных функций. Главная — **`QUERY`**: SQL-подобные запросы прямо в таблице. Плюс **`IMPORTRANGE`** (данные из другой таблицы), **`ARRAYFORMULA`** (формула сразу на весь столбец) и реальная совместная работа в реальном времени.
:::

## Зачем это нужно

Многие компании живут в Google Sheets, а не в Excel — из-за бесплатности и совместного доступа. Базовые формулы (`SUMIFS`, `VLOOKUP`, `XLOOKUP`) работают так же, но есть свои фишки, которые сильно ускоряют работу и которые любят спрашивать.

## QUERY — SQL внутри таблицы

`QUERY` позволяет писать почти-SQL по диапазону:

```text
=QUERY(A1:D; "select B, sum(D) where C='paid' group by B order by sum(D) desc")
```

Это полноценный `SELECT ... WHERE ... GROUP BY ... ORDER BY` по данным листа. Для аналитика, знающего SQL, — самый мощный инструмент Sheets.

:::note[Столбцы в QUERY — это буквы]
Внутри `QUERY` обращайся к столбцам по буквам листа (`A`, `B`, `C`), а не по заголовкам. `select B, sum(D)` — это столбцы B и D диапазона.
:::

## IMPORTRANGE — данные из другой таблицы

Подтягивает диапазон из другого Google-файла по ссылке:

```text
=IMPORTRANGE("URL_или_id_таблицы"; "Лист1!A1:D100")
```

При первом запуске нужно один раз разрешить доступ (кнопка «Allow access»). Удобно собирать данные из нескольких таблиц в одну сводную.

## ARRAYFORMULA — формула на весь столбец

Вместо протягивания формулы по тысячам строк — одна формула, работающая на весь массив:

```text
=ARRAYFORMULA(B2:B * C2:C)        перемножит столбцы целиком
```

Считается сразу для всего диапазона, не нужно копировать вниз. Часто комбинируют с `IF` и `IFERROR`.

## Прочие уникальные функции

- **`GOOGLEFINANCE`** — котировки и курсы валют: `=GOOGLEFINANCE("USDRUB")`.
- **`IMPORTHTML`** — таблица или список со страницы по URL.
- **`IMPORTDATA`** — данные из CSV/TSV по ссылке.

## Excel vs Google Sheets

| | Excel | Google Sheets |
|--|-------|---------------|
| Совместная работа | через OneDrive | в реальном времени, нативно |
| `QUERY` (SQL-стиль) | нет | есть |
| Мощность на больших данных | выше | предел ~10 млн ячеек, тормозит раньше |
| Power Query | есть | нет (вместо — `QUERY`/Apps Script) |
| Автоматизация | VBA | Apps Script (JavaScript) |

## Apps Script (обзор)

Для автоматизации в Sheets вместо VBA — **Apps Script** на JavaScript: можно писать функции, ставить триггеры (например, обновлять данные по расписанию), слать письма. На старте знать не обязательно — достаточно понимать, что такая возможность есть.

## Задачи для самопроверки

<details>
<summary>1. Нужна выручка по категориям с сортировкой — одной формулой в Sheets. Как?</summary>

`=QUERY(A1:D; "select B, sum(D) where C='paid' group by B order by sum(D) desc")`. `QUERY` заменяет связку фильтр + сводная одной строкой. Столбцы указываются буквами листа.

</details>

<details>
<summary>2. Когда Excel предпочтительнее Google Sheets?</summary>

На больших объёмах (сотни тысяч строк, тяжёлые модели), при работе с Power Query и когда нужна полноценная офлайн-работа. Google Sheets выигрывает в совместном доступе, бесплатности и функции `QUERY`.

</details>
