## Зачем это нужно

pandas — это надстройка над Python. Если не понимать базовый синтаксис (что такое список, как работает срез `[1:3]`, что делает `for`), код анализа будет магией. Здесь — ровно та база, что встречается в реальных ноутбуках.

## Типы данных

```python
n = 42          # int — целое
price = 19.99   # float — дробное
name = "Анна"   # str — строка
is_paid = True  # bool — True/False
nothing = None  # None — «ничего», аналог NULL
```

Проверить тип — `type(n)`. Преобразовать — `int("42")`, `str(42)`, `float("3.14")`.

## Коллекции

Четыре контейнера, два из которых используются постоянно:

```python
nums = [1, 2, 3]                 # list — упорядоченный, изменяемый
point = (55.7, 37.6)             # tuple — как list, но неизменяемый
user = {"name": "Анна", "age": 30}  # dict — пары ключ-значение
tags = {"ru", "paid"}            # set — уникальные значения
```

`list` и `dict` — рабочие лошадки. Словарь — это, по сути, строка таблицы: `user["name"]` → `"Анна"`.

## Срезы и индексация

Индексы с нуля; срез `[начало:конец]` не включает конец:

```python
s = [10, 20, 30, 40, 50]
s[0]      # 10  — первый
s[-1]     # 50  — последний
s[1:3]    # [20, 30]  — со 2-го по 3-й
s[:2]     # [10, 20]  — первые два
s[::2]    # [10, 30, 50] — каждый второй
```

Та же логика работает со строками и позже — с pandas. Стоит запомнить твёрдо.

## Условия и циклы

```python
amount = 2500
if amount > 2000:
    tier = "крупный"
elif amount > 1000:
    tier = "средний"
else:
    tier = "мелкий"

for x in [1, 2, 3]:
    print(x * 10)
```

Отступы (4 пробела) — часть синтаксиса, а не оформление: ими Python определяет границы блока.

## Функции и lambda

```python
def discount(price, pct):
    return price * (1 - pct / 100)

discount(1000, 20)        # 800.0

# lambda — короткая функция «на месте», часто внутри pandas
fee = lambda x: x * 0.05
```

## Comprehensions

Компактный способ построить список/словарь — встречается в коде анализа постоянно:

```python
squares = [x**2 for x in range(5)]              # [0, 1, 4, 9, 16]
paid = [o for o in orders if o > 2000]          # с условием (фильтр)
prices = {k: v*1.2 for k, v in base.items()}    # dict comprehension
```

Читается как «возьми x для каждого x в range, [если условие]».

## f-строки

Подстановка значений в строку — для подписей, логов, путей:

```python
country, total = "RU", 4300
print(f"Выручка {country}: {total} ₽")    # Выручка RU: 4300 ₽
print(f"Доля: {0.1234:.1%}")              # Доля: 12.3%
```

:::caution[Отступы и =/==]
Две классические ошибки новичка: смешивать табы и пробелы в отступах (Python ругнётся `IndentationError`) и путать `=` (присваивание) с `==` (сравнение). В условии всегда `==`.
:::

## Задачи для самопроверки

<details>
<summary>1. Что вернёт `[10, 20, 30, 40][1:3]`?</summary>

`[20, 30]`. Срез берёт элементы с индекса 1 включительно до индекса 3 не включая. Индексация с нуля, правая граница не входит.

</details>

<details>
<summary>2. Как одной строкой получить список квадратов чётных чисел от 0 до 9?</summary>

`[x**2 for x in range(10) if x % 2 == 0]` → `[0, 4, 16, 36, 64]`. Comprehension с условием `if`.

</details>
