:::tip[Коротко]
Новый столбец почти всегда считается **векторно**: `df["net"] = df["amount"] * 0.95` — быстро и читаемо. `apply` нужен лишь для сложной логики, `map` — для замены значений по словарю, `assign`/`pipe` — для аккуратных цепочек преобразований.
:::

## Зачем это нужно

Сырые данные редко содержат нужную метрику напрямую: надо посчитать налог, категоризировать суммы, привести валюту. Это и есть трансформации — создание и изменение столбцов.

```python title="Демо-DataFrame df"
#  order_id country status  amount
#  101      RU      paid    2500
#  104      KZ      paid    4200
#  106      DE      paid    3000
```

## Векторное создание столбцов

Самый частый и быстрый способ — арифметика прямо на столбцах:

```python
df["net"]   = df["amount"] * 0.95            # минус 5%
df["is_big"] = df["amount"] > 2000           # True/False
df["label"] = df["country"] + "-" + df["status"]   # конкатенация строк
```

:::caution[Не цикли по строкам]
Соблазн написать `for i in range(len(df)): ...` или `df.apply(..., axis=1)` для простой арифметики. Это в десятки раз медленнее векторной операции и хуже читается. Цикл/`apply` — только когда векторно никак.
:::

## apply — для сложной логики

Когда нужна функция, которую не выразить арифметикой:

```python
def tier(x):
    if x >= 3000: return "A"
    if x >= 2000: return "B"
    return "C"

df["tier"] = df["amount"].apply(tier)        # apply к Series
```

Категоризацию часто удобнее делать через `pd.cut(df["amount"], bins=[0,2000,3000,np.inf], labels=["C","B","A"])`.

## map — замена по словарю

`map` подменяет значения `Series` по словарю — удобно для перекодировки:

```python
names = {"RU": "Россия", "KZ": "Казахстан", "DE": "Германия"}
df["country_ru"] = df["country"].map(names)
```

## assign — столбцы в цепочке

`assign` возвращает новый DataFrame с добавленным столбцом — удобно в цепочках, не трогая оригинал:

```python
result = (
    df
    .assign(net=df["amount"] * 0.95)
    .assign(tax=lambda d: d["amount"] - d["net"])
)
```

## eval и pipe

```python
df.eval("net = amount * 0.95", inplace=True)   # арифметика строкой (быстро на больших df)

def add_tax(d, rate):
    return d.assign(tax=d["amount"] * rate)

df.pipe(add_tax, rate=0.2)                      # вставить свою функцию в цепочку
```

`pipe` позволяет встраивать собственные функции в «трубопровод» преобразований, сохраняя читаемый стиль.

## Когда что применять

| Задача | Инструмент |
|--------|-----------|
| Арифметика по столбцам | векторно: `df["a"] * 2` |
| Замена значений по справочнику | `map` |
| Сложная построчная логика | `apply` (или `np.select`/`pd.cut`) |
| Добавить столбец в цепочке | `assign` |
| Встроить свою функцию в цепочку | `pipe` |

## Задачи для самопроверки

<details>
<summary>1. Почему `df["net"] = df["amount"] * 0.95` лучше цикла по строкам?</summary>

Это векторная операция: pandas/NumPy применяют умножение ко всему столбцу разом на уровне C — в десятки-сотни раз быстрее цикла и короче в коде. Цикл по строкам в pandas — антипаттерн для простой арифметики.

</details>

<details>
<summary>2. Нужно заменить коды стран на полные названия. Какой инструмент?</summary>

`map` со словарём: `df["country"].map({"RU":"Россия", ...})`. Он создан ровно для поэлементной замены значений по соответствию ключ→значение.

</details>
