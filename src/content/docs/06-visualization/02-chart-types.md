:::note[В разработке]
Страница будет наполнена в ближайшее время.
:::

## План страницы

- Сравнение значений: bar, column
- Композиция: pie (когда можно), stacked bar, treemap
- Распределение: histogram, box, violin
- Взаимосвязь: scatter, heatmap
- Изменение во времени: line, area
- Иерархия: treemap, sunburst
- Шпаргалка-схема
