Выбор BI зависит от региона: в США доминирует Tableau, в СНГ и Европе — Power BI (после ухода Tableau из РФ). Looker Studio (бывший Google Data Studio) — бесплатный вариант для пет-проектов. Я фокусируюсь на двух главных, остальные — обзорно.

## Что внутри

- **Tableau** — подробный разбор
- **Power BI** — подробный разбор с DAX
- **Looker Studio** — бесплатная альтернатива
- **Metabase / Superset** — open-source
