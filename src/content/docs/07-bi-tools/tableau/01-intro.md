:::tip[Коротко]
Tableau — мощный BI-инструмент для интерактивных дашбордов, лидер рынка в США. Для обучения и портфолио бери **Tableau Public** (бесплатно). Логика работы: подключаешь данные → перетаскиваешь поля на полки (Rows/Columns) → Tableau строит график. Поля делятся на **измерения** (категории) и **меры** (числа).
:::

## Зачем это нужно

Дашборды просят почти в каждой вакансии аналитика. Tableau — один из двух главных инструментов (второй — [Power BI](/07-bi-tools/power-bi/01-intro/)). Освоив его, ты умеешь превращать таблицы в интерактивные отчёты, которые читает бизнес.

## Версии Tableau

| Версия | Что это | Цена |
|--------|---------|------|
| **Tableau Public** | Desktop с сохранением только в облако Public | бесплатно |
| **Tableau Desktop** | полноценная версия, локальное сохранение | платно |
| **Tableau Server** | self-hosted сервер для публикации внутри компании | платно |
| **Tableau Cloud (Online)** | то же, но в облаке Tableau | платно |

:::note[Бери Tableau Public для старта]
Для учёбы и портфолио Tableau Public закрывает почти всё: те же возможности построения, что у Desktop. Минус один — работы публикуются публично в облаке (нельзя хранить локально или приватно). Для пет-проекта это даже плюс — ссылку кладёшь в резюме.
:::

## Интерфейс

Ключевые элементы рабочего листа (worksheet):

- **Data pane** (слева) — поля источника, разделённые на **Dimensions** (измерения: страна, дата) и **Measures** (меры: сумма, количество).
- **Columns / Rows** — полки: перетащил поле → оно стало осью.
- **Marks card** — управление цветом, размером, меткой, типом графика.
- **Shelves Filters / Pages** — фильтры и постраничный просмотр.
- Вкладки снизу: Worksheets (графики), Dashboards, Stories.

## Измерения vs меры

Главная ментальная модель Tableau:

- **Dimension** (синие поля) — категории, по которым **режем** данные: страна, продукт, дата. Обычно идут в Rows/Columns и дробят результат.
- **Measure** (зелёные поля) — числа, которые **агрегируем**: сумма, среднее. По умолчанию Tableau их суммирует.

«Выручка (мера) по странам (измерение)» = перетащить `country` в Columns, `Sales` в Rows.

## Первое подключение

1. Запусти Tableau Public → **Connect** → выбери источник (Text file для CSV, Microsoft Excel, или сервер БД).
2. Tableau покажет вкладку **Data Source** — превью таблиц.
3. Перейди на лист (Sheet 1) — поля уже разнесены по Dimensions/Measures.
4. Перетащи поля на полки — график построится сам.

## Задачи для самопроверки

<details>
<summary>1. Какую версию Tableau взять для пет-проекта в резюме?</summary>

Tableau Public — бесплатна и достаточна по возможностям. Единственное ограничение: работы публикуются публично в облаке Tableau. Для портфолио это удобно — даёшь рекрутеру прямую ссылку на интерактивный дашборд.

</details>

<details>
<summary>2. Чем dimension отличается от measure?</summary>

Dimension — категориальное поле, по которому данные разрезаются на группы (страна, дата); measure — числовое поле, которое агрегируется (сумма, среднее). Грубо: измерения отвечают «в разрезе чего», меры — «что считаем».

</details>
